# 🍲 חומוס אליהו

אתר אינטרנט מודרני וחזותי עבור עסק של חומוס איכותי

---

## 📱 אודות הפרויקט

**חומוס אליהו** הוא אתר אינטרנט שלם למסעדה/ביסטרו המתמחה בחומוס טרי וטעים. האתר מוצע ממשק משתמש ידידותי ומעוצב בצורה מודרנית כדי להציג את המוצרים ולאפשר הזמנות.

### 🎯 המטרה

במהלך הדרך חקר אליהו מה הוא שומשום איכותי, ומתוך חקירה זו וגם קצת אבל חיפושים אחרים, הוא שכנע את עצמו שחומוס טוב זה לא קשה להכנה אם יודעים מה אתה עושה.

---

## ✨ תכונות עיקריות

- 🏠 **דף בית** - עם וידאו וגלריה תמונות, וטבלה החוזקת מיתוסים נפוצים על חומוס
- 📖 **דף אודות** - סיפור העלייה של חומוס אליהו
- 🍽️ **תפריט** - תצוגה של כל המנות הזמינות עם תמונות, תיאור וחיוך
- 📍 **סניפים** - מיקום כל הסניפים של חומוס אליהו
- 🛒 **הזמנות** - טופס הזמנה אינטרנטי נוח ופשוט
- ☰ **תפריט נייד** - תפריט יציב להנייטציה נוחה בכל גודל מסך

## 🛠️ טכנולוגיות

- **Responsive Design** - תאימות מושלמת לכל גודל מסך

## 🚀 התחלה מהירה

1. **פתח את הקבצים בדפדפן**
   ```bash
   # פשוט פתח את homePage.html בדפדפן שלך
   ```

2. **ניווט דרך האתר**
   - בחר מהתפריט העליון כדי להעבור בין העמודים
   - או השתמש בתפריט ההמבורגר (☰) בעיצוב נייד

3. **דרוג לצפוי במנות** ובחר "להזמנה" כדי הפנות הזמנה

## 🎨 עיצוב ומראה

- **צבע ראשי**: ירוק עמוק (#114800) - המשקף את ה"טביע" של חומוס איכותי
- **פונט**: Tel Aviv Brutalist עם fallback ל-Arial
- **תמונות**: תמונות באיכות גבוהה של המוצרים
- **ווידאו**: ווידאו טעינה/הדגמה בדף הבית
